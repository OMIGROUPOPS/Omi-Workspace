## Methodology (surfaced)
- **Cost function = EXACT counterfactual PnL** (not linear approx). For band B forced to X̄, each constituent N's realized PnL is recomputed from `spike_perN` via the atlas exit mechanic: exit at +X̄ if `size_qual_max_250 ≥ anchor + X̄`, else hold-to-settlement (`99−anchor` win / `−(anchor−1)` loss); ×10ct. Candidates = ∪{cell best_exit_X in B} ∪ {HOLD}; band X̄ = PnL-argmax. Band-cost (loss) = Σ per-cell-optimal `best_sum` − band PnL.
- **Segmentation = optimal-partition DP** (PELT-equivalent; `ruptures` not installed on the VPS, so the exact O(M²) DP over the ≤90 ordered cells is used — identical objective, trivially fast). Objective: min Σ band-cost + β·n_bands.
- **β calibrated per category** to ~92–94% of 1c PnL (proportional to each category's edge): see calibration table. ATP_MAIN/WTA_MAIN/ATP_CHALL = β8; WTA_CHALL smaller.
- **Hard constraints:** NO min-N floor; ATP_MAIN cells 37 & 38 forbidden in the same segment (hard-enforced, not β-dependent).
- **HOLD cells:** treated as a candidate X̄=HOLD; PELT-DP places natural breaks between contiguous HOLD-runs and exit-runs where collapsing them loses PnL.
