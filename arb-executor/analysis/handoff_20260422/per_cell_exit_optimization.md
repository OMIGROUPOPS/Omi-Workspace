# Per-Cell Exit Optimization
Generated: 2026-04-22 06:07:05 PM ET

## Methodology

- Pregame-close mid per side (volatility-jump detector, same as v2 heatmap)
- Settlement: ticker_meta.json settle_bid (>=80 = win = 100c, else = loss = 0c)
- For premarket_ticks (Apr 19-23): settlement from last tick (>=95c = win, <=5c = loss)
- Exit sweep: for each X in [3..35], check if max_bounce >= X
  - Hit: P&L = +X cents
  - Miss: P&L = (side_settle_price - entry_mid) cents
- Optimal X = argmax(mean_pnl). Tie-break on HR.
