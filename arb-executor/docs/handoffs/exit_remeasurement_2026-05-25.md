# Exit-X re-measurement — per-cell EV optimization (Sortino + bootstrap CI) — 2026-05-25

**Read-only** on the corpus tape (`{cat}_spike_perN.parquet`) — the same data the locked atlas + deployed adaptive bander were built on. Replaces the deployed bander's **raw-E[$] argmax** ("least-bad expected dollars") with **risk-adjusted per-cell selection**: bootstrap-CI'd mean PnL, downside-Sortino, a ≥50% hit-rate floor, and an explicit DISABLE for cells with no CI-supported edge. Cells are then re-banded by **price-adjacency run-length** on the recommended X (no minimum band size).

## 1. Methodology
- **PnL model (identical to the deployed bander — validated: recomputed deployed-$ matches the deployed band table to the dollar in all 4 categories).** Per constituent N: exit at +X if `size_qual_max_250 ≥ anchor + X`, else hold-to-settlement (win → `99−anchor`, loss → `−(anchor−1)`); HOLD = always hold. Total $ at 10ct = Σ(cents)·0.1.
- **Candidate X sweep:** {1,2,3,4,5,7,10,15,20,25,30,40,50,65,80, HOLD}.
- **Per (cell, X):** hit rate = P(`sq ≥ anchor+X`); mean PnL (¢/contract); total $ at 10ct; downside semivariance = mean(min(pnl,0)²) (target=0); **Sortino = mean / √(downside semivariance)**; Kelly sign = sign(mean); **bootstrap CI** = 2.5/97.5 pct of the mean over 1000 resamples of the per-N PnL.
- **Selection rule (as specified):** among candidates with **Sortino>0 AND CI-upper>0 AND (hit≥50% or HOLD)**, pick **max Sortino**. If none qualify → **DISABLE**. (Note: the rule keys on CI-*upper*>0, which is permissive; cells whose CI-*lower*<0 are kept but **flagged** as ci_straddle below.)
- **Re-banding:** adjacent price cells sharing the same recommended X (or both DISABLE) are merged; otherwise single-cell bands. No minimum band size.
